# ByteVerity eval kit — macOS (Apple Silicon)

Download **`byteverity-lineage-eval-macos-arm64.zip`** (above), open Terminal in the
folder you downloaded it to, and run:

```
unzip byteverity-lineage-eval-macos-arm64.zip
```

Enter the password when prompted (sent to you separately).

Then follow the guide:

```
open byteverity-lineage-eval/START_HERE.md
```

Everything runs locally and offline — no account, no server, no network.
